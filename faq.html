Frequently Asked Questions

Is VonCoin Opensource?

Yes. Voncoin is absolutely opensource. Since Voncoin is utility driven through InRoid technologies, therefore even the hardware is opensource architecture.

How VonCoin system works?

VonCoin system is 3 step process.

  • Task generation : Currency notes scanned from various devices(Cash handling machines) around the globe, capture value, space and time data and share with the Network(P2P). This creates MoneyChain.
  • Info generation : Miners verify the money chain block and confirm value, space and time information. The extracted information is voted by all in network and winning miner(s) are allowed to create cryptocurrency based on trust protocol of VonCoin. The fiat trail is stored in Info Chain.
  • Value storage : The cryptocurrency protocol stores the transaction and new block awards on Value Chain. As such, value Chain is storage of VonCoin currency.

How to mine VonCoins?

VonCoins are generated by creating block of fiat currency bills/notes which are tracked through InRoid Class Cash Handling Machines. Cash Counting Machines are one of the prime classes apart from Vending machines and Bank ATMs, teller machines, etc.

Is VonCoin secure?

VonCoin is architected on 3 Blockchain technology and trust protocols of the network. Therefore PoW/PoS type vulnerabilities are avoided. More generally, attacks like 51% are not possible on VonCoin network. All the task are generated from distributed and legitimate sources to meet the vision of fairness and transparency. And generated Info Chains and Value Chains cannot be run by any monopolistic approach. Thus, we can say that VonCoin is design-wise secure from vulnerabilities.

Why is currency tracking important?

For ages since fiat circulation in 15th century, fiat circulation has created valuable commerce and trade system. The fiat system brought efficiency and ease of doing business. But along came the vices. From the ages, theft and ransom of various forms have created an alternate balck economy which pushes inflation, recession and affordability with every such act of deciet. Currency tracking is an innovative solution, but doing this without P2P or such large network is very costly, time consuming and also trust-less if not done by all and together. Doing currency tracking on a P2P network model designed by Satoshi Nakamoto is a perfect task which is in sink with visions of cryptocurrency, free world and true step towards empowerment process.

Why is KYC required for task generating machines?

VonCoin is utility based to track money laundering and misuses of fiat currency. To keep the purpose legitimate and truly traceable, all task-generators will have to undergo KYC process and help law-enforcement agencies. We would always cooperate with all local laws for any such requirements.
